First Installation:

	Download a localhost server application. MAMP is recommended.

	Put all the files in one folder on the server.

	Open MySQL server with MAMP port specified
	Execute create_library_data_processed, remember to change the csv file paths to the correct path.
	Getting the Library database in your database system.

	Go to http://localhost:8888/LibraryManagementSystem/index.html, the application is started.

Later use:
	
	Open localhost server.

	Go to http://localhost:8888/LibraryManagementSystem/index.html, the application is started.
